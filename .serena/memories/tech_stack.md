# Tech Stack

## Frontend
- **Next.js 15** - React framework with App Router
- **React 19** - UI library with Server Components
- **TypeScript 5.7.2** - Type safety and development experience
- **TailwindCSS 3.4.17** - Utility-first CSS framework
- **HeroUI 2.7.8** - Component library (currently migrating to this)
- **Framer Motion 12.9.4** - Animation library

## Backend & Database
- **Supabase** - Backend-as-a-Service (PostgreSQL + Auth + Real-time)
- **Supabase SSR** - Server-side rendering support
- **Row Level Security (RLS)** - Data security at database level

## Development Tools
- **Biome 2.0.4** - Linting and formatting
- **Vitest 3.1.2** - Testing framework with jsdom
- **Knip 5.46.4** - Dead code detection
- **pnpm 10.6.2** - Package manager
- **Node.js 22.5.1** - Runtime environment

## Form Handling & Validation
- **Server Actions** - Form submissions without client JavaScript
- **Valibot 1.0.0** - Runtime schema validation

## UI Components & Icons
- **Heroicons** - Icon library
- **Tabler Icons** - Additional icon set
- **Lucide React** - More icons
- **Radix UI** - Headless UI components
- **QR Code React** - QR code generation

## Additional Libraries
- **date-fns 4.1.0** - Date manipulation
- **Recharts 2.15.3** - Chart components
- **DND Kit** - Drag and drop functionality
- **next-themes** - Theme switching
- **class-variance-authority** - Styling utilities
